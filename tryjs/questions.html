What is this?

This is a simple page to try javascript and jQuery.

You can start writing and executing code right now!

Write your code in the middle, log to the left using log(anyObject) o logJSON(anyObject) and run code with F9.

Why you made this thing?

Because I couldn't find anything simple enough, but not simpler.

Full IDEs like Cloud9 are incredible cool and very usefull to create a whole project, but way too complex when I just needed to test some JavaScript.

JSFIDDLE is excelent to create a whole page, but still a little too complex when I just needed to test some JavaScript.

The much simpler and notepad-like writecodeonline lacks jQuery support and having to type document.write for logging is too annoying.

TRYJS lays somewhere between writecodeonline and JSFIDDLE.

Hope you enjoy it!
